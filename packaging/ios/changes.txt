5.0:

Regina 5.0 is out!

This release includes 4-manifolds and normal hypersurfaces, tree
decompositions, simplification via the Pachner graph, and exact
Turaev-Viro invariants.  There are also many optimisations under the hood.

If you want more, you can download the desktop version from
regina.sourceforge.net – this includes basic support for manifolds of
dimension 5–15, and a major overhaul of the API for C++ and Python programmers.
