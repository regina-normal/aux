Regina for Ubuntu: Vivid (Ubuntu 15.04)

Ubuntu 15.04 (Vivid Vervet) already comes with Regina version 4.96. You can install it through the Ubuntu Software Centre as you would any other software package.

Since Ubuntu 15.04 has already passed its end-of-life, it is highly recommended that you upgrade to a newer version of Ubuntu.

There will be no updates for newer versions of Regina on Ubuntu 15.04.

The main package for Regina is called regina-normal. This is all that most people will need.

  • If you are a C++ programmer and you wish to build Regina into your own projects, you will also want the package regina-normal-dev.

  • If you want the MPI-enabled utilities for use on high-performance clusters, you will want the package regina-normal-mpi.
