Regina - Deprecation Guide

This page relates only to C++ and Python programmers.

Over the years, several outdated routines and classes within Regina's calculation engine have become deprecated, and have been marked for deletion at some future date.

That future date is now approaching. With the release of Regina 5.0, all of the deprecated code will be removed. If you use any of this deprecated code, your Python scripts or C++ programs will no longer work.

When will this change happen?

As of Regina 4.94, this process has now begun. Only a small amount of code has been removed so far, but more will be removed over time.

How do I fix my code?

The following table lists the routines and classes that will be (or have been) removed, as well as the replacements that you can use instead. Any changes in the behaviour of these routines or classes are noted in the comments column.

The blue cells indicate code that is still present (but deprecated). The red cells indicate code that has now been removed.
